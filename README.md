## üß† Project Description
AutoVision.AI is an intelligent computer vision system designed to detect and classify multiple objects in real-time for autonomous vehicles. The system applies **Deep Learning** models such as **YOLOv8** and **OpenCV** to recognize vehicles, pedestrians, and road elements, ensuring improved road safety and decision-making.  

This project focuses on integrating machine learning with real-world driving data to enable autonomous vehicles to interpret surroundings effectively.
